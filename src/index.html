This is a minimal webpage created by the DILNET SAs for quickly downloading certificates and config files for DILNET2.0, DILNET2.0 2.4Ghz, and eduroam.

DILNET2.0

  • Connection guide
  • Certificate file: upd_dilnet.crt
  • iOS and macOS config file: dilnettwo.mobileconfig

eduroam

  • Connection guide
  • Certificate file: upd_eduroam.crt
  • iOS and macOS config file: eduroam.mobileconfig

DILNET2.0 2.4Ghz

  • Usual instructions apply for Windows and Android, but change the network name/SSID to: DILNET2.0 2.4Ghz
  • iOS and macOS config file: dilnettwopointfour.mobileconfig
